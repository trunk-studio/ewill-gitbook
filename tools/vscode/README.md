### Mac 常用快速鍵

* Command + Shift + P 或 Ｆ1: 顯示所有命令 (Command Palette)
* Command + P: 前往檔案
* Command + Shift + E：檢視檔案總管
* Command + Shift + F：檢視搜尋
* Command + Shift + X：檢視擴充功能
* Command + F：尋找目前檔案內的字
* Shift + G: 移至第 N 行
* Control + \`：開啟終端機
* Control + Command + 空白鍵：開啟表情符號
* Command + `+`：放大
* Command + `-`：縮小
